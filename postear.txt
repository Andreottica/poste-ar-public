POSTE.AR - MANUAL DE USO
========================

Red social anónima con identidad persistente.
Sin registro, sin IPs, sin datos guardados.
Todo se borra cada lunes a las 00:00 hs.

¿QUÉ ES POSTE.AR?
-----------------
Un lugar para comunicarse sin dejar rastro.

A diferencia de WhatsApp, Telegram, Signal u otros:
- No pedimos número de teléfono
- No pedimos email
- No creás una cuenta
- No hay empresa que conozca tu identidad
- No hay perfil que te siga por la web
- Todo desaparece automáticamente cada lunes

Tu identidad la controlás vos con palabras que
solo vos conocés. Nada más.


TU IDENTIDAD ANÓNIMA
====================

CÓMO CREAR TU USUARIO
---------------------
Al entrar por primera vez:

1. Escribí 4-5 palabras (ej: perro mesa luz golf)
2. Presioná [ entrar ]
3. El sistema genera tu usuario anónimo (ej: meno.34.rite.87)

IMPORTANTE: Tus palabras SON tu identidad.
- Si las recordás → siempre recuperás el mismo usuario
- Si las olvidás → perdés tu identidad para siempre
- Si las compartís → otra persona puede publicar como vos

RECUPERAR TU USUARIO
--------------------
Para volver a ser el mismo usuario:

1. Ingresá las MISMAS palabras que usaste antes
2. El sistema genera el MISMO usuario automáticamente

CAMBIAR DE IDENTIDAD
---------------------
Presioná [ salir ] y entrá con palabras distintas.
Cada combinación de palabras genera un usuario único.

RECOMENDACIONES
---------------
- Usá 4-5 palabras fáciles de recordar
- Ejemplo: "gato verde tiza luna"
- Anotá tus palabras en papel (NO en computadora)
- Nunca uses menos de 3 palabras (muy inseguro)

RECORDAR SESIÓN
---------------
Al entrar podés tildar:
- "Recordar usuario": muestra tu usuario al volver
- "Recordar password": entra automáticamente al abrir

Si destildás una opción, se borra inmediatamente.
Para mayor seguridad, destildá todo antes de cerrar
en una computadora compartida.


DOS MODOS DE USO
================

Poste.ar tiene dos modos completamente separados.
El botón [ público ] / [ encriptado ] los alterna.

Cambiar de modo borra el contexto del otro modo.


MODO PÚBLICO
============

El timeline abierto. Todos pueden leer.

PUBLICAR
--------
1. Escribí tu mensaje (máximo 576 caracteres)
2. Presioná [ postear ]
3. Aparece en el timeline con tu usuario y color

FORMATO DE TEXTO
----------------
*texto*              → negrita
_texto_              → cursiva
<nombre:url.com>     → enlace clickeable
#hashtag             → etiqueta destacada

Ejemplo:
Esto es *importante* y _urgente_.
Más info: <aquí:ejemplo.com> #documentos

BUSCAR
------
El buscador encuentra mensajes en toda la base.

- Una palabra: "documentos"
- Varias palabras: "documentos, firmados"
- No distingue mayúsculas/minúsculas


MODO ENCRIPTADO
===============

Un canal cifrado invisible al timeline público.
Solo existe para quien tenga las palabras semilla.

Nadie puede saber que hay mensajes cifrados.
Nadie puede leerlos sin la semilla correcta.

PUBLICAR EN MODO ENCRIPTADO
----------------------------
1. Pasá a modo [ encriptado ]
2. Escribí tu mensaje (máximo 288 caracteres)
3. Escribí tu semilla (contraseña de grupo)
4. Presioná [ postear ]

El mensaje se cifra en TU navegador antes de enviarse.
El servidor recibe solo texto ilegible.

LEER MENSAJES CIFRADOS
-----------------------
1. Abajo de los botones aparece la sección "Leer"
2. Escribí la semilla compartida
3. Presioná [ leer ]
4. Aparecen TODOS los mensajes que coincidan

Si la semilla es incorrecta: no aparece nada.
No hay mensaje de error porque no hay nada que mostrar.

MANTENER SEMILLA
-----------------
Tildá "Mantener semillas" para no tener que
reescribir la semilla entre posteo y posteo.

Al volver a modo público se destilda automáticamente.

CONVERSACIONES GRUPALES CIFRADAS
---------------------------------
Cuando varias personas usan la MISMA semilla,
se crea un canal secreto compartido.

Ejemplo con semilla "gato verde luna":
- Ana publica mensaje cifrado
- Carlos publica mensaje cifrado
- María publica mensaje cifrado

Cuando cualquiera ingresa "gato verde luna":
→ Se ven TODOS los mensajes del grupo
→ En orden cronológico
→ Cada uno con su usuario y color

→ Sin rastro en el timeline público
→ Sin que nadie más sepa que existe ese canal

DIFERENCIA IMPORTANTE
---------------------
- Palabras de IDENTIDAD: crean tu usuario permanente
- Semilla de MENSAJES: cifra el contenido del canal

Son DOS cosas distintas. No las confundas.


HERRAMIENTAS
============

ACTUALIZAR
----------
Presioná [ actualizar ] para ver mensajes nuevos.

TEMAS Y MODO NOCHE
------------------
[ tema ] → Cambia el estilo visual (desktop)
[ día/noche ] → Alterna modo claro/oscuro

SALIR
-----
Presioná [ salir ] para cerrar tu sesión.
Tu identidad se borra del navegador.


SEGURIDAD Y PRIVACIDAD
======================

CIFRADO AES-256
---------------
Mismo estándar que bancos y apps de mensajería segura.

Con 4-5 palabras: romper el cifrado requiere años
de procesamiento con supercomputadoras.

CÓMO FUNCIONA
-------------
1. Escribís mensaje + semilla en TU navegador
2. TU navegador cifra el mensaje (AES-256)
3. Se envía al servidor SOLO la versión cifrada
4. El servidor guarda texto ilegible

El servidor NUNCA ve:
- Tus palabras de identidad
- Tus semillas de mensajes
- El contenido de mensajes cifrados

QUÉ NO GUARDAMOS
----------------
- Palabras de identidad
- Semillas de mensajes
- Direcciones IP
- Datos del navegador
- Cookies de rastreo
- Ubicación geográfica

QUÉ SÍ GUARDAMOS
----------------
- Mensajes públicos (texto plano)
- Mensajes cifrados (ilegibles para el servidor)
- Usuario generado (ej: meno.34.rite.87)
- Fecha y hora de publicación
- Color del usuario (generado automáticamente)

TODO SE BORRA CADA LUNES 00:00 HS
----------------------------------
Máximo de vida de un mensaje: 7 días.
El sitio muestra un contador con el tiempo restante.


LÍMITES TÉCNICOS
================

CARACTERES
----------
- Mensaje público: 576 caracteres máximo
- Mensaje cifrado: 288 caracteres máximo
- Palabras de identidad: 3-12 palabras
- Semilla de mensajes: 1-12 palabras (más = más seguro)

SIN EDICIÓN
-----------
No se pueden editar ni borrar mensajes.
Solo desaparecen el lunes 00:00hs.

SIN MULTIMEDIA
--------------
Solo texto. No se aceptan imágenes, videos ni archivos.


FILOSOFÍA
=========

En un mundo donde todo se registra y vende,
poste.ar es un espacio que se borra solo.

No hay algoritmos. No hay perfiles. No hay permanencia.
No hay empresa que venda tus datos.
No hay teléfono que te identifique.

Solo comunicación, aquí y ahora.

CÓDIGO ABIERTO
--------------
El código completo es público y auditable.
Presioná [ source ] para acceder al repositorio.

STACK TÉCNICO
-------------
- Frontend: HTML, CSS, JavaScript vanilla
- Backend: Node.js + Express
- Base de datos: LibSQL (SQLite remoto)
- Cifrado: CryptoJS (AES-256 + SHA-256)


ADVERTENCIAS
============

- Anotá tus palabras de identidad en PAPEL
- No publiques contenido ilegal
- El anonimato no te protege de consecuencias legales
- Si olvidás tus palabras de identidad: perdés tu usuario PARA SIEMPRE
- Si olvidás una semilla de mensaje: ese canal se pierde PARA SIEMPRE
- No hay recuperación posible (no guardamos nada)
